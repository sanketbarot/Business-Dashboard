# AI Toolcor Business Dashboard

> Smart business finance tracker for modern entrepreneurs

**Website:** [business.aitoolcor.com](https://business.aitoolcor.com)

---

## 📊 Overview

A powerful, privacy-first business finance dashboard that helps you track income, expenses, and profits with beautiful visualizations. All data is stored locally on your device — no server required.

---

## ✨ Features

### 📊 Dashboard
- Real-time financial overview
- Period-based analytics (Today / Week / Month / Year / All Time)
- Income vs Expense charts
- Expense breakdown by category
- 30-day cash flow trend
- Smart AI-powered insights
- Month-to-month comparison
- Top expense categories
- Payment mode analytics

### 💳 Transactions
- Add / Edit / Delete transactions
- Bulk selection & deletion
- 9 filter presets + custom date range
- Advanced multi-field filtering
- Table & Card view modes
- Full-text search
- Pagination
- Export to Excel & PDF
- JSON backup / restore

### 🎨 Design
- Midnight Aurora theme
- Cyan + Gold accents
- Fully responsive
- Smooth animations
- Modern typography

---

## 🚀 Tech Stack

- **HTML5 / CSS3 / JavaScript** (Vanilla)
- **Chart.js** — Data visualizations
- **SheetJS** — Excel export
- **jsPDF** — PDF export
- **LocalStorage** — Data persistence
- **PWA** — Install as app
